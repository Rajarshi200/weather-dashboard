#  Advanced Weather Dashboard

A real-time weather dashboard built with **Vanilla JavaScript**, featuring Glassmorphism UI, geolocation tracking, and interactive maps.

## ‚ú® Features
- **Real-time Data:** Fetches live weather metrics (Temp, Humidity, Wind) via OpenWeatherMap API.
- **Geospatial Visuals:** Interactive city maps powered by **Leaflet.js**.
- **Smart Geolocation:** One-click "Locate Me" feature using the Browser Geolocation API.
- **Data Persistence:** Saves recent search history using LocalStorage.
- **Dynamic UI:** Backgrounds adapt based on weather conditions (Rain/Clear/Clouds).
